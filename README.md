**AdventureWorks Sales Analytics | Power BI Project**

📌Descripción del Proyecto:

Este proyecto consiste en el desarrollo de un dashboard analítico en Power BI utilizando la base de datos de ejemplo AdventureWorks. El objetivo principal es transformar datos transaccionales de ventas en información útil para el análisis de negocio y la toma de decisiones.

El proyecto incluye:

- Modelado de datos en esquema estrella.

- Limpieza y transformación de datos con Power Query.

- Creación de métricas y KPIs utilizando DAX.

- Diseño de dashboards interactivos.

- Análisis de ventas, rentabilidad y rendimiento comercial.

🎯 Objetivos:

- Analizar el desempeño de ventas de la empresa.

- Identificar tendencias de crecimiento mensual y anual.

- Evaluar la rentabilidad por categorías y productos.

- Detectar los productos con mejor rendimiento.

- Facilitar la exploración de datos mediante visualizaciones interactivas

🛠️ Tecnologías Utilizadas:

- Microsoft Power Bi
- Power Query
- DAX
- SQL
- Modelo dimensional
